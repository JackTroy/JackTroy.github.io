  3. python 中的 == 与 is
    • Python中的对象包含三要素：id、type、value
    • 其中id用来唯一标识一个对象，type标识对象的类型，value是对象的值
    • is判断的是a对象是否就是b对象，是通过id来判断的
    • ==判断的是a对象的值是否和b对象的值相等，是通过value来判断的
  4. 'image{}.jpg'.format(0), format()函数起到替代作用, 参数替代字符串里的{}
