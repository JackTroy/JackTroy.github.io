• Following is the order in which SQL elements get executed as this will change the way we can optimize SELECT queries:

    1. FROM Clause
    2. ON
    3. OUTER
    4. WHERE
    5. GROUP BY
    6. HAVING
    7. SELECT
    8. DISTINCT
    9. ORDER BY
    10. TOP
